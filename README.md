# movienight

Next.js (app router) scaffold with Base Web + Styletron, Vitest, and Playwright.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`
- `npm run test:unit`
- `npm run test:e2e`
- `npm run format`

## Tooling

- Base Web + Styletron
- ESLint (next/core-web-vitals), Prettier
- Vitest + Testing Library (unit), Playwright (e2e)
- Husky + lint-staged (pre-commit) ; pre-push runs lint/typecheck/test:unit

## Env

- Copy `env.example` to `.env.local` and set `TMDB_API_KEY`.

No commits/pushes have been made for these changes.
